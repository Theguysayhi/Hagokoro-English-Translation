【編集時の注意点】
","、"@"、" [[[ "、"#"、"13"　などの半角記号や数字を書き足したり、削除したり、書き換える行為は
絶対に行わないでください。

これらの記号によってゲーム内でテキストを区切ったり処理を行ったりしています。
この記号が正しい配置になっていないと
ゲーム内でテキストの読み込みができず、ゲームの進行に
影響が出る可能性があります。


【各テキストファイルの説明】

[Dialogue_table.txt]
 -> エッチイベントで使用する敵のセリフが書かれています。
　　ここで書かれている文章は[Text_Event.txt]にて使用されます。


[Name_table.txt]
 -> キャラクターなどの名前が保管されているテキストファイルです。


[Text.txt]
 -> 今の所未使用です。製品版などで使用されることもあるかもしれません


[Text_enemyWord.txt]
 -> 戦闘時に使用される敵のセリフです。色仕掛け、空蝉の術などで使います


[Text_Event.txt]
 -> 各種通常イベント、エッチイベントの文章が入っています。
　このファイルの下のほうに各記号の説明を書いております。
　仕組みを熟知すればあなただけのエッチシナリオもできるかもしれません
　（開発中につき上手く動かなかったらすいません）

[Text_Item.txt]
 -> アイテム、武器、防具の名前や説明などが入っています。


[Text_Notice.txt]
 -> システム関連の詳細が書いているテキストファイルです。


[Text_QuestData.txt]
 -> 策戦任務などのテキストが入っています。


[Text_UI.txt]
 -> ボタンを始めとした各UIに使用する文字が入っています。



【各テキストファイルで使われている記号の説明】
"#"
->シャープ記号で文字データを一つずつ分けています。絶対に消したり足したりしないでください。


","
-> "#"で切り分けた文字をさらにこのカンマ記号で分けます。
　絶対に消したり足したりしないでください。


"/"
-> Text_Item.txtなどで使用。","で区切った文字をさらにこのスラッシュで切り分けます。
　絶対に消したり足したりしないでください。


"10005"、"501"、"51"などのそれぞれの文字の最初に書いてある数字
-> ゲーム内で使うインデックスです。
　絶対に消したり足したり書き換えたりしないでください。


"[ba]"、"[Tosyu]"などの [ ] で覆われた記述
-> ゲーム内で固有の単語に置き換えられるワードです。
　絶対に書き換えたり書き足したりしないでください。



【Text_Event.txt固有の記号】
"@"
-> Text_Event.txtなどで使用。
　読み込んだテキストをこのアットマーク「@」で区切り、それがゲーム内の
　メッセージウインドウで表示されるセリフひとつとなります。
　
"/"
-> Text_Item.txtなどと同じ記号ですが、こちらはこの左側に書いてある数字が
　セリフの名前欄表示に使われます。たとえばText_Event.txt内の

「13/三つの流派を操る侵入者…
　物見の報告は本当のようですね。@」

なら　「13」が発言者を表しています。この数字は「Name_Table.txt」と紐付いており、
そちらを見ると「美々」のセリフであると分かります。


"]]0016"　のような　" ]] "で始まっている行
-> 敵を表示させるための演出で使われています。
　絶対に書き換えたり消したりしないでください。


"[[020000"　のような　" [[ "で始まっている行
-> 暗転、射精、エッチ体制変更などの記述です。
　仕組みを熟知するまでは絶対に書き換えたり消したりしないでください。
　この下に各演出効果について書き記しています。


"]]]|W1|H1|t0|Y0.43|S0.20@"　のような　" ]]] "で始まっている行
-> カットインカメラを表示する命令です。
　"|"で区切られており、それぞれの数字に基づいたカットインが挿入されます。
　W1：カットインカメラの横幅を指定します。0～1で、"W0.3"などと入れると画面幅の3割ほどの大きさの
　　　カメラが表示されることになります。

　H1：カットインカメラの縦幅を指定します。0～1で、"H0.8"などと入れると画面の高さ8割ほどの
　　　カメラが表示されます。

　t0：注視する敵を指定します。　"t0"なら一番目の敵、　"t2"なら三番目の敵を注視する
　　　という形です。

　Y0.43：注視する高さを指定します。　たとえば"Y0.25"のように数字が低いほど下になり、
　　　"Y0.55"など数字が高いほど顔など上のほうに注目したカメラになります。

　S0.20：カットインカメラのズーム率を指定します。数字が小さいほどズームインします。
　　　　"S0.20"は結構アップになったズーム率です。　"S0.35"で普通ぐらいです


"[[[00003@"　のような " [[[ "で始まっている行
-> 背景を変更します。絶対に書き換えたり消したりしないでください。
